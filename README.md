## 📱 Cara Instalasi APK

### Langkah-Langkah Instalasi:

1. **📥 Download** file APK dari tombol di atas
2. **🔓 Aktifkan "Install dari sumber tidak dikenal"**
   - Buka Pengaturan → Keamanan → Unknown Sources
   - Aktifkan opsi tersebut
3. **📂 Buka File APK** yang sudah diunduh
4. **✅ Ikuti Proses Instalasi** yang muncul di layar
5. **🎉 Selesai!** Aplikasi siap digunakan

> ⚠️ **Catatan:** Jika diminta izin, klik "Allow" untuk melanjutkan instalasi

## 💻 Untuk Developer

### 📋 Prerequisites

Pastikan Anda sudah menginstall:
- **Node.js** (v16 atau lebih tinggi)
- **npm** atau **yarn**

### 🚀 Setup Lokal

#### 1. Clone Repository
```bash
git clone https://github.com/pundhy89/BaliTopTour.New.git
cd BaliTopTour.New
```

#### 2. Install Dependencies
```bash
npm install
```

#### 3. Setup Environment Variables
Buat file `.env.local` dan tambahkan:
```
GEMINI_API_KEY=your_api_key_here
```

Dapatkan API key gratis dari [Google AI Studio](https://aistudio.google.com/apikey)

#### 4. Run Development Server
```bash
npm run dev
```

#### 5. Buka di Browser
```
http://localhost:3000
```
