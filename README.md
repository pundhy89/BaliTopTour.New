# 🌴 Bali Top Tour

Aplikasi wisata Bali dengan AI-powered recommendations untuk menemukan destinasi terbaik di Bali.

**Website:** [https://balitoptour-new.vercel.app](https://balitoptour-new.vercel.app)

---

## 📥 Download APK

Unduh aplikasi Bali Top Tour langsung untuk Android:

📱 **[Download BaliTopTour.v3.apk](https://github.com/pundhy89/BaliTopTour.New/releases/download/v1.0.0/BaliTopTour.v3.apk)** (v1.0.0)

### Cara Install APK:
1. Download file APK dari link di atas
2. Aktifkan "Install dari sumber tidak dikenal" di pengaturan Android Anda
3. Buka file APK dan ikuti proses instalasi
4. Selesai! Nikmati aplikasi Bali Top Tour

---

## 🚀 Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key

3. Run the app:
   ```bash
   npm run dev
   ```
